# VVGankIO
A simple client for Gank.io.This Client now mainly contains the pictures of girls every day.

#用到的东西
- MVP
- Retrofit+OkHttp(封装)
- Glide(可以加载图片和动图，又比fresco小很多，所以选择的这个)
- 封装了BaseRecyclerAdapter
